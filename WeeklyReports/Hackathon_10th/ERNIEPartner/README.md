# 飞桨黑客松第十期·文心合作伙伴赛道 — 进阶任务周报

本目录用于存放**飞桨黑客松第十期·文心合作伙伴赛道**进阶任务认领者的周报记录。

> 本目录仅供**进阶任务**认领者使用（任务序号 18-34）。
> 打卡任务（序号 1-17）通过邮件提交截图，无需提交周报。

---



## 目录命名规则

```
任务序号_认领者GitHubID/
```

示例：GitHub ID 为 `dev_alice`，认领了进阶任务 #20（天数智芯）：

```
ERNIEPartner_20_dev_alice/
```

---

## 文件命名规则

```
[WeeklyReport]YYYY.mm.dd~YYYY.mm.dd.md
```

- 日期范围为本周的起止日期
- 文件名中**不含空格**
- 通过 **PR** 提交，每2周一次

示例：

```
[WeeklyReport]2026.04.21~2026.04.27.md
```

## 提交说明

1. 在 `WeeklyReports/Hackathon_10th/ERNIEPartner/` 下创建以 `ERNIEPartner_任务序号_GitHubID` 命名的子目录
2. 将周报文件放入该子目录
3. 通过 PR 提交至 `PFCCLab/Camp` 仓库
4. PR 标题格式：`[黑客松10th·文心伙伴] 周报 #任务序号 GitHubID YYYY.mm.dd`
